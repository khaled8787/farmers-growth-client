# Project Name: Farmer Crops Marketplace

A FullStack stack application where farmers can post their crops and buyers can show interest in buying them.


## Features
- Farmers can post crops with details like name, type, price, quantity, and location.
- Buyers can show interest in crops.
- Farmers can view, edit, or delete their crops.
- Farmers can manage buyer interests (accept/reject).
- User authentication using Firebase.
